## June 25, 2024

- Added function callng support  (public beta ) to select models, see here: https://octo.ai/docs/documentation/text-gen-solution/function-calling-using-octo-ai-models
- Added support for creating fine tunes on OpenPipe and exporting them to OctoAI for low-cost inference:
 https://octo.ai/docs/text-gen-solution/openpipe-loras

- Added WizardLM2-8x22B, a fine tun of Mixtral-8x22 from Microsoft.
- Removed Qwen1.5B and Hermes-2-Pro-Mistral-7B

## June 12, 2024

Added the ability to self-serve downgrade from Pro to Trial tier.
